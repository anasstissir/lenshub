# EY Visual Identity Reference

Apply these standards to every slide generated by ey-slide.

---

## Brand Palette

| Name            | Hex       | Usage |
|-----------------|-----------|-------|
| EY Yellow       | `#FFE600` | Primary accent, CTA boxes, active quadrant highlights, chart primary bars |
| EY Dark Grey    | `#2E2E38` | Body text, header backgrounds, titles, spine lines |
| EY Mid Grey     | `#747480` | Secondary text, footnotes, axis labels |
| EY Light Grey   | `#D8D8D8` | Inactive elements, alternating row fills, borders |
| EY Off-White    | `#F5F5F5` | Alternating table row fill |
| EY Green        | `#00A859` | Positive indicators, opportunities, waterfall gains |
| EY Amber        | `#FFB81C` | Warning indicators, medium-risk items |
| EY Red          | `#E4002B` | Negative indicators, threats, waterfall losses |
| EY Blue         | `#0070C0` | Links, process arrows, secondary accent |

## Slide Layout

### Dimensions
- **Slide size:** Widescreen 16:9 — 25.4cm × 14.29cm (or 10in × 5.63in in pptxgenjs)
- **Margins:** 0.5in all sides

### Standard Zones
```
┌─────────────────────────────────────────────────┐
│ [EY YELLOW BAR — 0.25in]  logo top-right        │
│─────────────────────────────────────────────────│
│                                                 │
│  TITLE ZONE (y=0.3–1.1in)                       │
│  Full-width, 22pt bold, #2E2E38                 │
│                                                 │
│  CONTENT ZONE (y=1.2–5.0in)                     │
│  Varies by framework — see frameworks.md        │
│                                                 │
│─────────────────────────────────────────────────│
│ [FOOTER — source left, page number right, 8pt] │
└─────────────────────────────────────────────────┘
```

### Cover Slide Layout
- Full-width EY Yellow band (top 35% of slide)
- Deck title: 28pt Bold, white text on yellow
- Subtitle / date / audience: 16pt Regular, EY Dark Grey
- EY logo: top-right corner, white version on yellow
- Confidentiality notice: bottom-left, 8pt, #747480

### Section Divider Layout
- Left half: EY Yellow fill
- Right half: White
- Section name: 24pt Bold, EY Dark Grey, centred vertically on right half
- Section number: Large (72pt), EY Yellow opacity 30%, right half background

## pptxgenjs Bootstrap

Use this boilerplate for every generated `slides.ts`:

```typescript
import pptxgen from 'pptxgenjs';

const pptx = new pptxgen();

// Slide dimensions (widescreen 16:9)
pptx.defineLayout({ name: 'WIDESCREEN', width: 10, height: 5.63 });
pptx.layout = 'WIDESCREEN';

// EY brand colours
const EY = {
  yellow:    'FFE600',
  darkGrey:  '2E2E38',
  midGrey:   '747480',
  lightGrey: 'D8D8D8',
  offWhite:  'F5F5F5',
  green:     '00A859',
  amber:     'FFB81C',
  red:       'E4002B',
  blue:      '0070C0',
};

// Slide master helper
function addSlideChrome(slide: pptxgen.Slide, title: string, pageNum: number, total: number) {
  // EY Yellow top bar
  slide.addShape(pptx.ShapeType.rect, { x: 0, y: 0, w: 10, h: 0.18, fill: { color: EY.yellow } });
  // Title
  slide.addText(title, {
    x: 0.5, y: 0.28, w: 8.5, h: 0.7,
    fontSize: 20, bold: true, color: EY.darkGrey, fontFace: 'Calibri',
  });
  // Footer bar
  slide.addShape(pptx.ShapeType.rect, { x: 0, y: 5.43, w: 10, h: 0.2, fill: { color: EY.lightGrey } });
  // Page number
  slide.addText(`${pageNum} / ${total}`, {
    x: 8.5, y: 5.43, w: 1.4, h: 0.2, fontSize: 8, align: 'right', color: EY.midGrey,
  });
}
```

## Chart Style Rules

- **Bar charts:** EY Yellow for primary series; EY Mid Grey for comparison series; no 3D effects; no gradients
- **Line charts:** EY Dark Grey primary line (2pt); EY Yellow accent markers; gridlines in EY Light Grey (0.5pt)
- **Pie/Donut:** EY Yellow largest segment; clockwise from top; labels outside with leader lines
- **Waterfall:** Positive = EY Yellow; Negative = EY Red; Total bars = EY Dark Grey
- **All charts:** White background; no chart border; no legend box border; data labels in #2E2E38

## Slide Quality Checklist

Before finalising any slide:
- [ ] Action title is a declarative sentence (insight, not topic label)
- [ ] EY Yellow bar present at top; footer present at bottom
- [ ] No more than 5 bullets per slide
- [ ] Font stack: EY Interstate / Calibri / Arial — no other fonts
- [ ] Colours from EY palette only — no custom colours
- [ ] All charts: white background, no 3D, no gradients
- [ ] Source/footnote present if any external data cited
- [ ] Confidential watermark if deck is marked confidential
